# Digital-Storage-Oscilloscope


## Overview

This project is a portable **Digital Storage Oscilloscope (DSO)** built using an **ESP32 (38-pin)** and a **TFT display**. The system acquires analog signals, processes them in real time, and displays the waveform on the TFT screen. A custom PCB was designed and soldered for the oscilloscope and charging module, making the device compact and portable.

---

## Features

- Real-time waveform visualization
- Signal acquisition from 20 Hz to several kHz
- Rechargeable portable design
- Custom PCB implementation
- TFT graphical display

---

## Hardware Used

- ESP32 (38-pin)
- TFT SPI Display
- Li-ion Battery
- Custom Charging Module
- Custom PCB
- Passive Components

---

## Software Used

- Arduino IDE
- Embedded C/C++

---

## Working

1. The analog input signal is sampled using the ESP32 ADC.
2. The firmware processes the sampled data.
3. The waveform is displayed on the TFT screen in real time.
4. The rechargeable battery powers the complete system through the custom charging module.
